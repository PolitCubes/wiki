# Работы

Работы используются для получения внутриигровой валюты, в виде монет

Проверить свой баланс можно при помощи команды:

`/bal`

Для того, чтобы устроиться на работу, пропишите команду:

`/jobs browse`

У Вас откроется меню с выбором доступных работ

![jobs1](/economy/jobs1.png)

Наведитесь на выбранную работу, чтобы узнать краткие характеристики:

- Монеты за выполнение работы
- Опыт для перехода на следующий уровень
- Количество рабочих на данной работе
- Действия для получения монет

![jobs2](/economy/jobs2.png)

Список доступных работ

    Строитель - Получайте деньги за постройку структур
    Фермер - Получайте деньги за выращивание культур
    Шахтер - Получайте деньги за выкапывание минералов и руд
    Дровосек - Получайте деньги за рубку деревьев

Нажмите **ЛКМ** для подробной информации о действиях, за которые можно получить валюту

Нажмите **ПКМ**, чтобы устроиться на выбранную работу

> Также используйте **ПКМ** для того, чтобы уволиться с работы


При выполнении работы, за каждое действие Вы получаете опыт и впоследствии уровень.

Повышая каждый уровень, процент от получаемых монет увеличивается на 1

![jobs3](/economy/jobs3.png)

Максимальный уровень работ: 200

> Каждый новый уровень стоит дороже предыдущего

![jobs4](/economy/jobs4.png)


Вперед зарабатывать монеты!
