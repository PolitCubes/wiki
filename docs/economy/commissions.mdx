# Комиссии и эмбарго

## **Установление комиссий**
 
Комиссии открывают широкие возможности для ведения битв не только на полях сражений, но и в сфере торговли.
 
Лишь лидер нации способен установить комиссии по отношению к другим городам и нациям
>`/n set commission [nation/town] [нация/город] [процент]` *(но не более 25%)*

:::note Примечание!

- Можно установить комиссию внутри нации, взиматься будет один раз.

- Все деньги с комиссий будут идти в казну нации

:::

После установления комиссий при попытке перевода будет всплывать такое уведомление:
![comission1](/economy/comission1.avif)

![comission3](/economy/comission3.avif)

:::note Примечание!

При взаимодействии с честшопом комиссию платит тот, кто взаимодействует с честшопом. При продаже ресурсов клиент получает меньше, при покупке отдаст больше, чем изначально указал продавец. 

:::

Лидер нации может установить базовую комиссию, которая будет применяться по отношению ко всем игрокам вне зависимости от их принадлежности(включая Вашу) к какой-либо нации:
 
 >`/n set commission default [процент]`
 
Любой игрок может посмотреть комиссию определенной нации или города (без параметров покажет комиссии вашей нации):

>`/commission check [nation/town/default] [нация/город]`

:::note Примечание!

На сервере действует следующий приоритет комиссий:

- Первым идет проверка комиссии на **город**(town)

- Вторым идет проверка комиссии на **нацию**(nation)

- Третьим идет проверка комиссии **по умолчанию**(default)

:::

## **Эмбарго**

Лидер нации также может объявить эмбарго:

>`/n set embargo [nation/town] [нация/город] [on/off]`
 
При объявлении эмбарго нация/город не сможет:

-   Переводить деньги командой `/pay` людям из нации/города, на которую наложено эмбарго
    
-   Использовать в любом месте честшопы людей из нации, на которую наложено эмбарго.
    
-   Использовать любые честшопы на территории города, находящегося в нации с которым действует эмбарго.
