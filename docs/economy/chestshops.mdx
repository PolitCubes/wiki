# Честшопы

Для того, чтобы торговать в автономном режиме Вам нужно будет использовать механику «честшопов».

Иными словами, сундук, на который поставили табличку.

## **Создание магазина на продажу**

1. Пропустите первую строку. *(там будет ваш ник)*
2. Во второй строке напишите **[cs]** или **[ChestShop]**
3. В третьей строке укажите тип магазина и его цену

> Например: **S 25**, где *("s" = sell)*, а *"25"* - количество монет.

4. Пропустите четвертую строку.

![cs_s1](/economy/cs_s1.avif)

5. После создания честшопа нажмите **ПКМ** предметом, который Вы продаете, по табличке.

Далее откроется меню, где можно выбрать количество товара:

![cs_s2](/economy/cs_s2.avif)

Проверить магазин можно, нажав **ПКМ** по табличке с уже готовым честшопом:

![cs_s3](/economy/cs_s3.avif)

## **Создание магазина на покупку**

1. Пропустите первую строку. *(там будет ваш ник)*
2. Во второй строке напишите **[cs]** или **[ChestShop]**
3. В третьей строке укажите тип магазина и его цену

> Например: **B 25**, где *("b" = buy)*, а *"25"* - количество монет.

4. Пропустите четвертую строку.

![cs_b1](/economy/cs_b1.avif)

5. После создания честшопа нажмите **ЛКМ** предметом, который Вы покупаете, по табличке.

Далее откроется меню, где можно выбрать количество товара:

![cs_b2](/economy/cs_b2.avif)

Проверить магазин можно, нажав **ПКМ** по табличке с уже готовым честшопом:

![cs_b3](/economy/cs_b3.avif)

## **Создание магазина на обмен**

1. Пропустите первую строку. *(там будет ваш ник)*
2. Во второй строке напишите **[cs]** или **[ChestShop]**
3. Пропустите третью строку.
4. Пропустите четвертую строку.

![trade1](/economy/trade1.avif)

5. После создания честшопа нажмите **ЛКМ** предметом, который Вы продаете, а затем **ПКМ** предметом, который Вы покупаете.

>Выберите количество товара для продажи

![trade2](/economy/trade2.avif)

>Выберите количество товара для покупки

![trade3](/economy/trade3.avif)

Проверить магазин можно, нажав **ПКМ** по табличке с уже готовым честшопом:

![trade4](/economy/trade4.avif)

В игровой чат придет сообщение об успешном обмене:

![trade5](/economy/trade5.avif)
