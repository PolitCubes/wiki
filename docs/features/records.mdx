# Пластинки

Кастомные пластинки тоже являются неотъемлемой частью нашего сервера, добавляющей какую-то интересную изюминку в игровой процесс.

## **Новые пластинки и способ их получения**

Создаем Виниловый рекордер, который будет генерировать новые пластинки.

![records1](/features/records1.jpg)

Вместе с этим нам будут нужны пустые пластинки:

![records2](/features/records2.jpg)

Кликаем пластинкой по рекордеру, помещаем в него 5 алмазов, ждем 60 секунд. После завершения работы, кликаем правой кнопкой мыши по рекордеру

![records3](/features/records3.jpg)

![records4](/features/records4.jpg)

После получения пластинки вам остается лишь вставить её в проигрыватель, и наслаждаться музыкой!

![records5](/features/records5.jpg)
