# Пластинки

Кастомные пластинки являются неотъемлемой частью нашего сервера, добавляя интересную изюминку в игровой процесс.

## **Пластинки и способ их получения**

Создайте проигрыватель в количестве двух штук.

Для этого Вам потребуется:

- 18 досок
- 2 алмаза

> Один проигрыватель понадобится для создания винилового рекордера, а второй для воспроизведения пластинки.

![records1](/features/records1.png)

Создайте "Виниловый рекордер", который будет генерировать случайные пластинки.

![records2](/features/records2.png)

После создания и установки Винилового рекордера, в чат поступит уведомление:

![records3](/features/records3.png)

Вместе с этим Вам нужно создать пустые пластинки:

Для этого Вам потребуется:

- 4 черных красителя
- 1 золото

![records4](/features/records4.png)

Кликните пластинкой по рекордеру, поместите в него 5 алмазов, ожидайте 60 секунд.

![records5](/features/records5.png)

После завершения работы, кликните **ПКМ** по рекордеру. У Вас появится сгенерированная случайным образом пластинка.

![records6](/features/records6.png)

После получения пластинки Вам останется лишь поместить ее в проигрыватель и наслаждаться музыкой!

![records7](/features/records7.png)
