# Честшопы

Для того, чтобы торговать в автономном режиме вам нужно будет использовать механику «честшопов». Иными словами, сундук, на который поставили табличку.

## **Создание честшопа**

### **Создание магазина на продажу**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. В третьей **S x**, где *("s" = sell)*, а *"х"* - количество монет.
4. Четвертую строку **пропускаете**.

![cs1](/features/cs1.jpg)

5. После создания честшопа нужно будет нажать **ПКМ** предметом, который Вы продаете, по табличке. Далее откроется меню, где можно выбрать количество товара:

![cs2](/features/cs2.jpg)

### **Создание магазина на покупку**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. В третьей **B x**, где *("b" = buy)*, а *"х"* - количество монет.
4. Четвертую строку **пропускаете**.

![cs3](/features/cs3.jpg)

5. После создания честшопа нужно будет нажать **ЛКМ** предметом, который Вы покупаете, по табличке. Далее откроется меню, где можно выбрать количество товара:

![cs4](/features/cs4.jpg)

### **Создание магазина на обмен**

1. Первую строку **пропускаете**. *(там будет ваш ник)*
2. Во второй пишите **[cs]** или **[ChestShop]**
3. Третью строку **пропускаете**.
4. Четвертую строку **пропускаете**.

![cs5](/features/cs5.jpg)

5. После создания честшопа нужно будет нажать **ЛКМ** предметом, который Вы продаете, а затем **ПКМ** предметом, который Вы покупаете.


### **Взаимодействие игроков с готовым честшопом**

* Для того, чтобы убедиться в правильности, Вы можете нажать **ПКМ** по табличке с уже готовым честшопом:

![cs6](/features/cs6.jpg)

* Если продаваемый *(с точки зрения покупателя)* предмет не установлен, то идет продажа первого предмета, который продавец положил в сундук. 

> Таким образом можно без проблем продавать алкоголь и другие предметы.
