# Уникальные рецепты

На нашем проекте изменены рецепты некоторых стандартных(ванильных) предметов майнкрафт.

Ниже приведен список измененных предметов:

### **Губка**

![sponge](/features/recipes/sponge.avif)

## Предметы

### **Паутина**

![cobweb](/features/recipes/cobweb.avif)

![enchanted_golden_apple](/features/recipes/enchanted_golden_apple.avif)
