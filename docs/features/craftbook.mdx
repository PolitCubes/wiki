# Ворота, лифты, мосты

:::info Внимание!

Все механизмы в текущей статье совместимы с редстоуном

:::

## **Создание ворот**

1. Найдите место для постройки ворот и определите будущую **"рамку"** (*абсолютно любой формы*):

![gates1](/features/gates1.avif)

2. Заполните рамку одним из следующих **материалов**: *обычная стеклянная панель, железные прутья, адский забор, все виды деревянных заборов*

![gates2](/features/gates2.avif)

3. Поставьте табличку на стене, во второй строке напишите **[Gate]**

![gates3](/features/gates3.avif)

4. Ворота готовы и находятся полностью в вашем распоряжении!

![gates4](/features/gates4.avif)

![gates5](/features/gates5.avif)

## **Создание лифтов**

Для того, чтобы создать лифт, Вам потребуются только таблички и этажи.

> Данный механизм является весьма удобным и рациональным способом сокращения пути.

Для создания лифта, используйте следующие таблички:

* Предназначена для поднятия на этаж **выше**: *[Lift Up] на второй строчке*

![elevator1](/features/elevator1.avif)

* Предназначена для спуска на этаж **ниже**: *[Lift Down] на второй строчке*

![elevator2](/features/elevator2.avif)

* Работает исключительно как точка прибытия для предыдущих двух вариантов и не исполняет более никаких функций

![elevator1](/features/elevator3.avif)

*на изображении — две таблички [Lift Up] в одном ряду*

![elevator1](/features/elevator4.avif)


## **Создание мостов**

1. Для начала необходимо построить сам мост.

> Максимальная длина моста составляет 16 блоков, а ширина 5 блоков

![bridge1](/features/bridge1.avif)

2. Поставьте табличку на любую поверхность моста, во **второй строке** напишите **[Bridge]**

![bridge2](/features/bridge2.avif)

3. Поставьте такую же табличку на другом конце моста

![bridge3](/features/bridge3.avif)

4. Мост готов и находится полностью в вашем распоряжении!

![bridge4](/features/bridge4.avif)
