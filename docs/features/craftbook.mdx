# Лифты, ворота, мосты

:::info Внимание!

Все механизмы в текущей статье совместимы с редстоуном

:::

## **Создание ворот**

1. Найдите место для постройки ворот и определите будущую **"рамку"** (*абсолютно любой формы*):

![gates1](/features/gates1.png)

2. Заполните рамку одним из следующих **материалов**: *обычная стеклянная панель, железные прутья, адский забор, все виды деревянных заборов*

![gates2](/features/gates2.png)

3. Ставим табличку на стене, во **второй строке** пишем __*[Gate]*__

![gates3](/features/gates3.png)

4. Ворота готовы и находятся полностью в вашем распоряжении!

![gates4](/features/gates4.png)

![gates5](/features/gates5.png)

## **Создание лифтов**

Для того, чтобы создать лифт, Вам потребуются только таблички и этажи.
Данный механизм является весьма удобным и рациональным способом сокращения пути.

Для создания лифта, используйте следующие таблички:

* Предназначена для поднятия на этаж **выше**: *[Lift up] на второй строчке*

![elevator1](/features/elevator1.png)

* Предназначена для спуска на этаж **ниже**: *[Lift down] на второй строчке*

![elevator2](/features/elevator2.png)

* Работает исключительно как точка прибытия для предыдущих двух вариантов и не исполняет более никаких функций

![elevator1](/features/elevator3.png)

*на изображении — две таблички [Lift Up] в одном ряду*

![elevator1](/features/elevator4.png)


## **Создание мостов**

1. Для начала необходимо построить сам мост. Максимальная длинна моста 16 блоков, ширина 5 

![bridge1](/features/bridge1.png)
2. Ставим табличку, во **второй строке** пишем **[Bridge]**

![bridge2](/features/bridge2.png)
3. Делаем это с двух сторон моста 

![bridge3](/features/bridge3.png)
4. Можем пользоваться мостиком 

![bridge4](/features/bridge4.png)
