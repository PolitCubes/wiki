# Лифты, ворота, мосты

:::info Внимание!

Все механизмы в текущей статье совместимы с редстоуном

:::

## **Создание ворот**

1. Постройте **рамку** будущих ворот (*абсолютно любой формы*):

![craftbook1](/features/craftbook1.jpg)

2. Заполните рамку одним из следующих __материалов__: *обычная стеклянная панель, железные прутья, адский забор, все виды деревянных заборов*

![craftbook2](/features/craftbook2.jpg)

3. Ставим таблицу на рамке, во **второй строке** пишем __*[Gate]*__

![craftbook3](/features/craftbook3.jpg)

4. Ворота готовы и находятся полностью в вашем распоряжении!

![craftbook4](/features/craftbook4.jpg)

## **Создание лифтов**

Для того, чтобы создать лифт вам потребуются только таблички и этажи.
Является весьма удобным и рациональным способом сокращения пути.

Для создания лифта, используйте следующие таблички:

* Предназначена для поднятия на этаж **выше** *([Lift up] на второй строчке)*

![craftbook5](/features/craftbook5.jpg)

* Предназначена для спуска на этаж **ниже** *([Lift up] на второй строчке)*

![craftbook6](/features/craftbook6.jpg)

* Работает исключительно как точка прибытия для предыдущих двух вариантов и не исполняет более никаких функций

![craftbook7](/features/craftbook7.jpg)

*на изображении — две таблички [Lift Up] в одном ряду*

![craftbook8](/features/craftbook8.jpg)


## **Создание мостов**

1. Для начала необходимо построить сам мост. Максимальная длинна моста 16 блоков, ширина 5 

![bridge1](/features/bridge1.png)
2. Ставим таблицу на рамке, во **второй строке** пишем **[Bridge]** 

![bridge2](/features/bridge2.png)
3. Делаем это с двух сторон моста 

![bridge3](/features/bridge3.png)
4. Можем пользоваться мостиком 

![bridge4](/features/bridge4.png)
