# Скины и плащи

## **Установка скинов и плащей**

Для того, чтобы установить обычный скин или плащ формата 64х64 пикселей, авторизуйтесь в [личном кабинете](https://politcubes.com/cabinet) и нажмите на кнопку «*Установить*»

![skins1](/general/skins1.avif)

![skins2](/general/skins2.avif)

Нажмите на кнопку «*Установить скин\плащ*» и загрузите файл в формате **PNG**.
>Файл должен быть расширения 64х64 пикселей, а если Вам требуется HD скин — расширение файла должно рассчитываться по формуле 64*чётное число (64, 128, 256, 512, 1024) пикселей.

Исходник стандартного скина: [(скачать)](/general/sample-skin.png)

Шаблон для создания скинов: [(скачать)](/general/skin-template.png)

Для редактирования скинов, можете использовать любой графический редактор(Paint.NET, Adobe Photoshop и т.п) или использовать онлайн-редактор:

![skins3](/general/skins3.avif)

Также используйте графический или онлайн редактор для создания плаща:
