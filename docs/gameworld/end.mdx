# Энд

Край(Энд) — это темное космоподобное измерение, которое представляет собой множество островов, целиком состоящих из камня Края.

Когда персонаж перемещается в это измерение, он появляется на центральном острове, окружённом другими небольшими.
> Расстояние между центральным островом и дальними составляет примерно 1000 блоков — между ними нет ничего, кроме пустоты.

В Крае обитают: странники края, дракон и шалкеры.

В этом измерении генерируются все доступные биомы Края.

Странники находятся на всех островах, дракон на центральном, а шалкеры — в крепостях. Однако любые существа, кроме боссов, могут переместиться в Край с помощью портала.

>Размер измерения составляет 20000x20000 блоков

:::note Примечание!

Иногда администрация в качестве жеста доброй воли принимает решение о расширении Энда

:::

## **На проекте есть следующие особенности в измерении края**

- Генерация натуральных структур

- Всегда и везде включено пвп

- Нельзя создать город, но можно сделать аванпост командой: `/town outpost`

- Отключены кровати из обычного мира

- В крепостях увеличен шанс выпадения книги с зачарованием "Починка"

- Существует только один портал в это измерение


## Мероприятие по открытию портала в измерение Края

> *Первое время после вайпа(обычно 1-2 недели) измерение Края недоступно для игроков.*

Расположение портала определяется администрацией вручную. Открытие локации с порталом происходит посредством отдельного мероприятия.

Обычно это происходит по следующему сценарию:

1. Публикуется тизер с зашифрованной информацией о местоположении портала в группе Вконтакте или иных площадках
2. Игроки по информации из тизера решают небольшую загадку в игровом мире с координатами построенной локации, где находится портал
